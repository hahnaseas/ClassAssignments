What is jQuery? An Instructional Site on jQuery

In short, jQuery is a JavaScript library and was created to make it much easier to use JavaScript. It simplifies many of JavaScripts basic functions that include but aren't limited to such as HTML/DOM/CSS manipulation, AJAX, HTML event methods, and effects and animation.

Resize();
 This is one of the events that can trigger within your page DOM. With resize(), you can attach a handler function to the browser - in which when a user resizes their browser. This example will display the window's current width and height when you try to resize it by dragging its corners.
