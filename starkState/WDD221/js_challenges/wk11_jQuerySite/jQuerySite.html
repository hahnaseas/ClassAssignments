What is jQuery? An Instructional Site on jQuery:

In short, jQuery is a JavaScript library and was created to make it much easier to use JavaScript. It simplifies many of JavaScript's basic functions including, but not limited to, HTML/DOM/CSS manipulation, AJAX, HTML event methods, and effects and animation.


Two Examples Using jQuery:

  1. Click Method - Click()
  2. Keypress Method - Keypress()

Example One: Click();

This method attaches an event handler to elements for the event and is executed when the user clicks on the element and hides paragraph elements.

Click me!


Example Two: Keypress();

This method attaches an event handler function to elements that when executed, the browser displays a message when the keypress event is pressed, as well as how many times it is entered.


Keypress: 0
